void invokeMethod ( in co::Any  instance,
in co::MethodInfo  mi,
in co::Any[]  args,
out co::Any  returnValue 
) throws co::UnsupportedOperationException, co::IllegalArgumentException, co::MissingInputException, co::IllegalCastException

Invokes a method from an interface or native class instance.

Method arguments are passed in the args array, in the same order they are expected by the callee. If args contains more arguments than the number of method parameters, the excess arguments are ignored. However, if args contains fewer arguments than the number of parameters, a MissingInputException is raised.

Method return values are obtainable through the returnValue argument. Notice that you must always pass a Any at this parameter, even when the callee is a void method (in this case, the returnValue parameter will return unmodified).

Exceptions:
UnsupportedOperationExceptionif type is not a MethodContainer.
IllegalArgumentExceptionif instance is not an instance of type, or if the method mi does not belong to this type.
MissingInputExceptionif args contains fewer elements than the number of parameters expected by the callee.
IllegalCastExceptionif one of the arguments in args has no valid conversion to the actual parameter type. Please refer to Any's documentation for more information about type conversion rules.

co::Interface newProxy ( in co::DynamicProxyHandler  handler ) throws co::UnsupportedOperationException, co::IllegalArgumentException

Creates a proxy interface that forwards all calls to a given DynamicProxyHandler.

A dynamic proxy is a lightweight interface instance (i.e. it is not a real component instance) that forwards all its calls to a handler interface, or to the handler's component instance.

Exceptions:
UnsupportedOperationExceptionif type is not an InterfaceType.
IllegalArgumentExceptionif the passed handler is invalid (e.g. null).
