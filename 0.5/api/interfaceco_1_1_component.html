Member Function Documentation

void bindInterface ( in co::InterfaceInfo  receptacle,
in co::Interface  instance 
) throws co::NoSuchInterfaceException, co::IllegalArgumentException

Binds an interface instance to one of the component's receptacles.

If an interface instance is already bound to the receptacle, it is replaced by the new instance.

Exceptions:
NoSuchInterfaceExceptionif receptacle is not a valid receptacle of this component.
IllegalArgumentExceptionif the passed instance is incompatible with the receptacle.
co::Interface getInterface ( in co::InterfaceInfo  itfInfo ) throws co::NoSuchInterfaceException

Retrieves the interface instance "provided by" or "bound to" one of the component's interfaces.

If itfInfo is a facet, this method returns the corresponding interface instance provided by this component. Otherwise, if itfInfo is a receptacle, this method returns the interface instance that is currently bound to the receptacle (in this case, the result can be null if nothing is bound to the receptacle).

Exceptions:
NoSuchInterfaceExceptionif itfInfo is not a valid interface of this component.
T* getFacet (  ) [inline]

getFacet<T>(): convenience method that returns the instance of a facet 'T' provided by this component. Please notice that this method only works for statically-defined components. It will not retrieve proxy interfaces from dynamic components (such as those implemented in Lua).
