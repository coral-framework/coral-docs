Detailed Description

template<class T>
class co::RefVector< T >

An enhanced std::vector of co::RefPtr's.

Template Parameters:
Tan interface type (co::Interface).

Supports automatic coercion to co::ArrayRange< T* >. Also offers extra methods for sorted arrays: sortedInsert() and sortedFind().

Performs a binary search on the sorted range [first,last] using a Comparator function.

Returns true if a matching element is found, or false if the key is not in the sorted range. In either case, the parameter pos is set to the index where the element should be located in the sorted range.

Note:
The Comparator function is defined as follows: 
 int Comparator( const T* element, const Key& key )
  • returns zero when element == key
  • returns <0 when element < key
  • returns >0 when element > key
Warning:
Assumes the specified range is already sorted according to the Comparator function.
