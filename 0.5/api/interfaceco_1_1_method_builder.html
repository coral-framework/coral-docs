Detailed Description

Works in conjunction with a TypeBuilder to define methods for new types.

Member Function Documentation

void createMethod (  ) throws co::IllegalNameException, co::MissingInputException

Uses the data provided thus far to add a method to the TypeBuilder that owns this MethodBuilder.

Exceptions:
UnsupportedOperationExceptionif the parent type was already created.
MemberClashExceptionif another member with a clashing name was added to the type since this builder's creation.
MissingInputExceptionif the reference to the TypeBuilder that owns this MethodBuilder is null.
void defineException ( in co::ExceptionType  exceptionType ) throws co::IllegalArgumentException

Adds an exception type that can be raised by the method.

Exceptions:

IllegalArgumentExceptionif the passed exceptionType is invalid (e.g. null).
