Detailed Description

A generic builder of Coral Types.

Member Function Documentation

co::Type createType (  ) throws co::MissingInputException

Fills the Type with all the definitions set and returns it.

This method should be called only once and after all definitions have been set.

Exceptions:
UnsupportedOperationExceptionif the type is already created.
MissingInputExceptionif not all required definitions were set for this builder's type kind.
void defineAttribute ( in string  name,
in co::Type  type,
in bool  isReadOnly 
) throws co::UnsupportedOperationException, co::IllegalNameException, co::IllegalArgumentException

Adds an attribute to a StructType, NativeClassType or InterfaceType.

Exceptions:
UnsupportedOperationExceptionif the builder's kind is neither TK_STRUCT, TK_NATIVECLASS nor TK_INTERFACE or if the type was already created.
IllegalNameExceptionif the passed name is not a valid identifier.
IllegalArgumentExceptionif the passed type is invalid (e.g. null or the type is a component or exception) or if the passed isReadOnly modifier is true and the builder's kind is TK_STRUCT (structs are always writable).
IllegalNameExceptionif the passed name clashes with other members.
void defineIdentifier ( in string  name ) throws co::UnsupportedOperationException, co::IllegalNameException
