bool co::binarySearch ( Range< T >  range,
const Key &  key,
Comparator  compare,
size_t &  pos 
)

Performs a binary search on a sorted range using a comparison function.

Returns true if a matching element is found, or false if the key is not in the sorted range. In either case, the parameter pos is set to the index where the element should be located in the sorted range.

The search relies on a key-based comparison function, which is more flexible (and often more efficient) than element-based STL comparators.

The comparison function is defined is follows: 

 int compare( const Key& key, const T& element )
  • returns zero when key == element
  • returns <0 when key < element
  • returns >0 when key > element
Warning:
The range must already be sorted according to the comparison function.
