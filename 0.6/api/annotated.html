IArrayType representing a unidimensional array of elements
IClassTypeInterface for types that contain fields and methods
IComponentThe blueprint of a component
ICompositeTypeInterface for types that contain named members
IDynamicServiceProviderInterface for components that provide a dynamic set of services through dynamic service proxies
IEnumType representing an enumeration of constant identifiers, in the range [0, n]
IExceptionType representing a named exception that can be raised by methods
IFieldDescribes a field/property as a member of an IRecordType
IInterfaceType representing an interface
IllegalArgumentExceptionRaised to indicate that a method has been passed an illegal or inappropriate argument
IllegalCastExceptionRaised to indicate an invalid casting or conversion
IllegalNameExceptionRaised to indicate that a method has been passed an illegal name
IllegalStateExceptionRaised to indicate that something is in an unexpected or invalid state
IMemberInterface for the members of a composite type
IMethodOffers type introspection for a method, as a member of a IClassType
IMethodBuilderWorks in conjunction with an ITypeBuilder to define methods for new types
IModuleA set of custom extensions loaded from a namespace in the Coral path
IModuleManagerManager responsible for loading and maintaining the life cycle of all modules
IModulePartControls the life-cycle of an abstract artifact (such as a dynamic library or a script bundle) loaded as part of a module
IModulePartLoaderInterface responsible for recognizing and loading an IModulePart from a module's dir
INamespaceNamespaces contain types and child namespaces and are the inner nodes of the type repository tree
INativeClassType representing a native class (a C++ class wrapped as a Coral value)
IObjectBasic service provided by all components through the implicit "object" facet
IParameterDescribes a method parameter
IPortDescribes an interface (either a facet or receptacle) as part of a component
IRecordTypeInterface for types that contain fields
IReflectorProvides reflective programming facilities for a type
IServiceThe co.IService interface, implicitly inherited by all interfaces
IServiceManagerManages the system's services database, providing contextualized service lookups
IStructAn ordered set of named fields of various types
ISystemCoral's bootstrap interface (a singleton)
ITypeBasic type interface
ITypeBuilderA generic builder of Coral Types
ITypeManagerManages the database of loaded types, providing methods to load and create new types
ITypeTransactionAllows sets of interdependent types to be created atomically
LexicalUtilsLexical utility functions in a static class
MissingInputExceptionRaised to indicate that some input is still missing in order to complete an operation
MissingServiceExceptionRaised to indicate that a required service is not available
ModuleLoadExceptionRaised to indicate an error while loading a module
NoSuchPortExceptionRaised to indicate that the requested port is not available
NotSupportedExceptionRaised to indicate that the requested operation is not supported
OSPortable OS functions in a static class
Range< T >A higher-level iterator for one-dimensional arrays
RefPtr< T >A smart-pointer that implements reference counting for Coral services
RefVector< T >An enhanced std::vector of co::RefPtr's
TypeLoadExceptionRaised to indicate an error while loading a type
UuidA 128-bit Universally Unique IDentifier (UUID)
