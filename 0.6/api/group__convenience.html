IType* co::getType ( const std::string &  fullName)

Utility function to retrieve or load a type by name. This is equivalent to calling: 

 co::getSystem()->getTypes()->getType( fullName )

Please refer to co::ITypeManager::getType() for the list of exceptions this function may throw.

IObject* co::newInstance ( const std::string &  fullName)

Utility function to instantiate a component given its full type name. This is equivalent to calling: 

 co::getType( fullName )->getReflector()->newInstance()

Please note that this function may raise all exceptions raised by the aforementioned methods.

IService* co::getServiceForType ( IInterface *  serviceType,
IInterface *  clientType 
)

Utility function to get the best provider of serviceType for clients of type clientType. If clientType is null this function will retrieve the service's global instance. Please refer to co::IServiceManager::getServiceForType() for the list of exceptions this function may throw.

IService* co::getServiceForInstance ( IInterface *  serviceType,
IService *  clientInstance 
)

Utility function to get the best provider of serviceType for the given clientInstance. This is equivalent to calling: 

 co::getSystem()->getServices()->getServiceForInstance()

Please refer to co::IServiceManager::getServiceForInstance() for the list of exceptions this function may throw.

T* co::getService ( ) [inline]

Template function to get a global service by its type. Please refer to co::IServiceManager::getService() for the list of exceptions this function may throw.

T* co::getService ( co::IInterface *  clientType) [inline]

Template function to get a clientType-specialized service by its type. This picks the most appropriate service instance available for clients of the given clientType. Please refer to co::IServiceManager::getServiceForType() for the list of exceptions this function may throw.

T* co::getService ( co::IService *  clientInstance) [inline]

Template function to get a clientInstance-specialized service by its type. This picks the most appropriate service instance available for the given clientInstance. Please refer to co::IServiceManager::getServiceForInstance() for the list of exceptions this function may throw.
