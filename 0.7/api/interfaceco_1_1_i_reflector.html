void invoke ( in co::Any  instance,
in co::IMethod  method,
in co::Any[]  args,
out co::Any  returnValue 
) throws co::NotSupportedException, co::IllegalArgumentException, co::MissingInputException, co::IllegalCastException

Invokes a method from an interface or native class instance.

Method arguments are passed in the args array, in the same order they are expected by the callee. If args contains more arguments than the number of method parameters, the excess arguments are ignored. However, if args contains fewer arguments than the number of parameters, a MissingInputException is raised.

Method return values are obtainable through the returnValue argument. Notice that you must always pass a co::Any at this parameter, even when the callee is a void method (in this case, the returnValue parameter will return unmodified).

Exceptions:
NotSupportedExceptionif the type is not a class type.
IllegalArgumentExceptionif instance is not an instance of type.
IllegalArgumentExceptionif the method does not belong to this type.
MissingInputExceptionif args contains fewer elements than the number of method parameters.
IllegalCastExceptionif one of the arguments in args has no valid conversion to the actual parameter type. Please refer to co::Any's documentation for more about type conversion rules.
co::IService newDynamicProxy ( in co::IDynamicServiceProvider  dynamicProvider) throws co::NotSupportedException, co::IllegalArgumentException

Creates a dynamic service proxy that forwards all calls to a given dynamicProvider.

A dynamic service proxy is a lightweight object (i.e. it is not a full component instance) that forwards all its calls to a dynamicProvider.

Exceptions:
NotSupportedExceptionif the type is not an interface.
IllegalArgumentExceptionif the dynamicProvider is invalid (e.g. null).
