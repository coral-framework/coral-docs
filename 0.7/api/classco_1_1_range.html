Detailed Description

template<typename T>
class co::Range< T >

A generic iterator for one-dimensional arrays.

Template Parameters:
Tany acceptable element type for a C++ array.

The range [start, end) is represented by two pointers. The first and the last elements can be obtained by getFirst() and getLast(), respectively. A traversal can be made in either direction using popFirst() or popLast(). It's also possible to index an array element directly using operator[]. Testing a range is equivalent to testing whether it isEmpty().

Here's an example of how to iterate over a range: 

        std::vector<int> myArray;
        myArray.push_back( 3 );
        myArray.push_back( 5 );
        myArray.push_back( 8 );
        myArray.push_back( 13 );

        for( co::Range<int> r( myArray ); r; r.popFirst() )
        {
            print( "%i", r.getFirst() );
        }
Reference
The concept of range as an alternative to iterators was proposed by Andrei Alexandrescu in his talk "Iterators Must Go". Since arrays are Coral's only native collection type, we provide only this very simple Range class.
