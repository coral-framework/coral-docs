Detailed Description

Basic service provided by all components through the implicit "object" facet.

Member Function Documentation

co::IService getServiceAt ( in co::IPort  port) throws co::NoSuchPortException

Returns the service "provided by" or "bound to" one of this object's ports.

If port is a facet, this method returns the corresponding service provided by this component instance. Otherwise, if port is a receptacle, this method returns the service currently bound to the receptacle (null indicates no service is bound).

Exceptions:
NoSuchPortExceptionif port is invalid.
