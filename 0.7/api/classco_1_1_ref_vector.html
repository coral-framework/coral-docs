Detailed Description

template<class T>
class co::RefVector< T >

An enhanced std::vector of co::RefPtr's.

Template Parameters:
Ta service type (co::IService).

Supports automatic coercion to co::Range< T* >. Also offers extra methods for sorted vectors.
