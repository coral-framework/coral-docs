Detailed Description

A generic builder of Coral Types.

Member Function Documentation

co::IType createType ( ) throws co::MissingInputException

Finishes building the type and returns it.

This method must be called only once, after all definitions have been set.

Exceptions:
NotSupportedExceptionif the builder's type was already created.
MissingInputExceptionif not all required definitions were set for this builder's type.
void defineBaseType ( in co::IType  baseType) throws co::NotSupportedException, co::IllegalArgumentException

Makes a type inherit from a baseType.

Exceptions:
NotSupportedExceptionif the builder's type was already created, or does not support inheritance.
NotSupportedExceptionif a base type was already defined.
IllegalArgumentExceptionif baseType is invalid.

void defineIdentifier ( in string  name) throws co::NotSupportedException, co::IllegalNameException

Adds an identifier to an enum.

Values are assigned implicitly: the first identifier's value is zero, the second identifier's value is one, and so forth.

Exceptions:
NotSupportedExceptionif the builder's type was already created, or is not an enum.
IllegalNameExceptionif name is not a valid identifier or clashes with other members.
